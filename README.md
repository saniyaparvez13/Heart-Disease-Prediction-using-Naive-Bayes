# Heart-Disease-Prediction-using-Naive-Bayes
 This project applies a Gaussian Naive Bayes classifier to predict the presence of heart disease based on patient medical attributes.

# Dataset

The dataset contains medical information about patients, where the target variable indicates whether a person has heart disease or not.

Target column:
1 → Heart disease present
0 → No heart disease

# Objective

The goal of this project is to build a simple machine learning model that can classify whether a patient is likely to have heart disease using physiological and clinical features.

# Model Used
Gaussian Naive Bayes
Gaussian Naive Bayes is a probabilistic classification algorithm based on Bayes’ theorem. It assumes that features follow a normal (Gaussian) distribution and is commonly used for baseline classification problems due to its simplicity and speed.

# Evaluation Metrics

The model is evaluated using the following metrics:

Accuracy – Measures overall correctness of predictions
Precision – Measures correctness of positive predictions
Recall – Measures ability to detect actual positive cases .

# Results
The model produces predictions on the test dataset, and performance is assessed using the metrics above. Results may vary depending on the dataset split and preprocessing.
